(i) rand()
Usage: Generates a random integer.

Example: rand(1, 100) returns a random number between 1 and 100 (inclusive).


echo rand(1, 10); // Output: random number between 1 and 10


(ii) abs()
Usage: Returns the absolute (non-negative) value of a number.

Example: abs(-5) returns 5.

echo abs(-15); // Output: 15


(iii) str_replace()
Usage: Replaces all occurrences of a search string with a replacement string in another string.

Example: Replace "apple" with "orange" in a sentence.

echo str_replace("apple", "orange", "I like apple."); // Output: I like orange.

(v) ceil()
Usage: Rounds a floating-point number up to the nearest integer.

Example: ceil(4.2) returns 5.

echo ceil(7.1); // Output: 8
