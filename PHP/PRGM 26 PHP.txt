 Construct a php program to create a chess board in PHP using the “For” loop.



<!DOCTYPE html>
<html>
<head>
    <title>PHP Chess Board</title>
    <style>
        table {
            border-collapse: collapse;
        }
        td {
            width: 60px;
            height: 60px;
            text-align: center;
        }
        .white {
            background-color: #f0d9b5;
        }
        .black {
            background-color: #b58863;
        }
    </style>
</head>
<body>

<h2>Chess Board</h2>

<table border="1">
<?php
for ($row = 1; $row <= 8; $row++) {
    echo "<tr>";
    for ($col = 1; $col <= 8; $col++) {
        $total = $row + $col;
        if ($total % 2 == 0) {
            echo "<td class='white'></td>";
        } else {
            echo "<td class='black'></td>";
        }
    }
    echo "</tr>";
}
?>
</table>

</body>
</html>


OUTPUT:
A visually accurate 8x8 chess board with alternating black and white squares.
